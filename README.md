# Flutter Product App

A Flutter application that fetches products from a REST API and displays them with search, category filtering, pagination, and a detailed product view.

## Features

- 📦 Fetch products from DummyJSON API
- 🔍 Search products by title
- 🏷️ Filter products by category
- 📄 View detailed product information
- ♾️ Infinite scroll pagination

## Tech Stack

- Flutter
- Dart
- HTTP Package

## API Used

https://dummyjson.com/products

## Project Structure

```
lib/
├── models/
├── screens/
├── services/
├── widgets/
└── main.dart
```

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/bhavyasrivastava014/flutter-product-app.git
```

2. Navigate to the project

```bash
cd flutter-product-app
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
